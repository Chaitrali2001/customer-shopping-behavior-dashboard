# Customer Shopping Behavior Dashboard

## Project Overview
This project analyzes customer shopping behavior using Power BI and Python.

## Tools Used
- Power BI
- Python
- Pandas
- Matplotlib

## Key Insights
- Total Sales: $233,081
- Total Customers: 3,900
- Average Purchase: $59.76
- Clothing category generated the highest revenue
- PayPal was the most used payment method

## Dashboard Features
- KPI Cards
- Sales by Season
- Gender Distribution
- Payment Method Analysis
- Interactive Filtering
